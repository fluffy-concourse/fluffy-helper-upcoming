## How to Conribute
1. Fork the repo - click the "Fork" button on Github
2. Clone the repo <br>
Linux
```
sh git clone https://github.com/fluffy-concourse/helper-bot-python-edition.git
cd helper-bot
```
Windows
```
git clone https://github.com/fluffy-concourse/helper-bot-python-edition.git
cd helper-bot
```

3. Set up your evironment - Follow the setup guide in [SETUP_EVIRONMENT.md](https://github.com/fluffy-concourse/helper-bot-python-edition/blob/main/SETUP_EVIRONMENT).. or do what you want for your evironment.
4. Create a new branch <br>
Linux: ```sh git checkout -b feature-name``` <br>
Windows: ```git checkout -b feature-name```
5. Make changes and commit - Keep the commits small and descriptive.. and simple, I'm a stoopid autistic fomx
6. Submit a pull request - Open a pull request and describe your changes.

## Reporting issues
- Use **Github Issues** to report bugs or suggest features.
- Check if your issue already exists before creating a new one.
- Use appropriate labels.

## Terms
- **Do not claim the code or assets as your own.**
- **Do not sell the code or assets.**
- **Releasing modified code is fine, but proper credit must be given.**
